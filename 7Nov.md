## Docker

- Images
- Containers
- Volumes
- Networking

**Running a container and EC2 Instance**

* docker run hello-world - Creates a new container

- Container is always created from images.

- Images can be local or remote.

Docker client <--> Docker daemon <--> Docker Hub

- Docker client is the CLI tool that we use to interact with the Docker daemon.

- Docker Daemon - Background processes that run on the host machine.

- Docker Hub - Remote repository of images.
